# python 30 日小專題訓練  
感謝chatgpt協助設計題目 - 30日計畫：

Day1. 實現一個計算器程序，接受用戶輸入兩個數字和一個操作符（例如+、-、*、/），並輸出計算結果。
Day2. 實現一個程式，判斷給定的整數是否為質數。
Day3. 編寫一個程式，輸入一個整數 n，輸出費波納契數列的前 n 項。
Day4. 編寫一個程式，輸入一個字符串，判斷它是不是迴文字符串。
Day5. 編寫一個程式，將兩個列表合併為一個，並按升序排序。
Day6. 編寫一個程式，計算給定列表中所有元素的平均值。
Day7. 編寫一個程式，檢查一個字符串是否是另一個字符串的子串。
Day8. 編寫一個程式，接受用戶輸入的英文句子，統計其中每個單詞出現的次數。
Day9. 編寫一個程式，計算並列印出給定列表中的所有奇數。
Day10. 編寫一個程式，將一個十進制數轉換為二進制表示形式。
Day11. 編寫一個程式，接受用戶輸入的溫度（攝氏度），並將其轉換為華氏度。
Day12. 編寫一個程式，接受用戶輸入的字符串，去除其中的空格並將其逆序輸出。
Day13. 編寫一個程式，輸入一個列表，返回其中出現次數最多的元素。
Day14. 編寫一個程式，找出給定列表中的所有唯一元素。
Day15. 編寫一個程式，計算一個字符串中大寫字母和小寫字母的數量。
Day16. 編寫一個程式，接受用戶輸入的整數列表，計算列表中相鄰元素的差值，並返回差值列表。
Day17. 編寫一個程式，接受用戶輸入的半徑，計算並輸出圓的面積和周長。
Day18. 編寫一個程式，找出一個列表中所有的偶數並將其平方後輸出。
Day19. 編寫一個程式，接受用戶輸入的一系列數字，然後計算它們的和與平均值。
Day20. 編寫一個程式，接受用戶輸入的數字列表，將列表中的數字按升序排序後輸出。
Day21. 編寫一個程式，接受用戶輸入的字符串列表，將列表中的字符串連接成一個長字符串並輸出。
Day22. 編寫一個程式，接受用戶輸入的兩個字符串，檢查第一個字符串是否為第二個字符串的字母異位詞。
Day23. 編寫一個程式，接受用戶輸入的整數列表，輸出列表中所有元素的乘積。
Day24. 編寫一個程式，接受用戶輸入的一段文本，統計其中各個字母出現的次數並輸出。
Day25. 編寫一個程式，實現一個簡單的登錄系統，用戶輸入用戶名和密碼，程式判斷是否匹配。
Day26. 編寫一個程式，接受用戶輸入的字符串，將其中的元音字母替換為大寫後輸出。
Day27. 編寫一個程式，輸入一個整數 n，判斷它是否是完全數（即等於其所有因子之和）。
Day28. 編寫一個程式，接受用戶輸入的一段文本，統計其中的單詞數並輸出。
Day29. 編寫一個程式，接受用戶輸入的字符串和子字符串，統計子字符串在字符串中出現的次數。
Day30. 編寫一個程式，接受用戶輸入的一個字符串，判斷其中是否含有重複的字符。
